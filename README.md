# hirebolt_v2

Hirebolt marketing site — Next.js + Tailwind CSS, deployed to Cloudflare Pages.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy on Cloudflare Pages

This project is built for Cloudflare Pages via [`@opennextjs/cloudflare`](https://opennext.js.org/cloudflare).

```bash
npm run preview   # build + run locally in the Cloudflare Workers runtime
npm run deploy    # build + deploy to Cloudflare Pages
```
